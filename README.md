# Fake News Headline Classifier
Fake News Headline Classifier is a machine learning based web application that analyzes news headlines or statements and predicts whether they are real or fake using Natural Language Processing (NLP) techniques.

## Features
- Classifies news headlines or statements as Real or Fake
- Uses Machine Learning and NLP for text analysis
- Supports single or multiple text inputs
- Fast prediction results
- Simple and user-friendly web interface
- Pre-trained model with saved vectorizer
- Built using Python and Flask

## Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML
- CSS

## How to Run
1. Install required libraries
pip install flask scikit-learn pandas numpy

2. Run project
python detection.py

3. Open browser
http://127.0.0.1:5000/

## How It Works
1. User enters a news headline or statement
2. Text is cleaned and transformed into numerical features
3. Vectorizer processes the text
4. Trained model predicts Real or Fake
5. Result is displayed instantly
